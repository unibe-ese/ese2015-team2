ESET2 - Tutoren suchen und finden!

Vielen Dank, dass Sie ESET2 nutzen. Folgende Übersicht soll Ihnen eine Hilfe sein, um die Plattform effizient zu nutzen und um mit Studenten und Tutoren in Verbindung zu treten.

1. Profil bearbeiten

Statten Sie Ihr Profil mit einem Profilbild und einer Beschreibung zu Ihrer Person aus!

Dies macht Ihr Profil für Studenten und Tutoren attraktiver und kann ermutigen, Kontakt mit Ihnen aufzunehmen!

2. Vorlesungen suchen

Über "Vorlesungen" können Sie alle Vorlesungen einsehen, die auf ESET2 angeboten werden. Von dort können Sie Vorlesungen direkt in Ihr Profil aufnehmen.

Als Tutor signalisieren Sie damit, dass Sie für die Vorlesung Nachhilfe anbieten - als Student, dass Sie Nachhilfe suchen.

3. Tutoren suchen

Sie können Tutoren direkt über den Menupunkt "Tutoren suchen" ausfindig machen oder über eine Vorlesung.

Sobald Sie einen Tutoren/Tutorin gefunden haben, können Sie ihn/sie über das Profil kontaktieren.
